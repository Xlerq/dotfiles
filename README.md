# Dotfiles

My personal Arch Linux + Hyprland setup.

This repository contains my user configs, scripts, package lists, and selected system configuration used to recreate my desktop after a fresh install.

## Included

- Hyprland
- Waybar
- Foot
- Yazi
- Helix
- Cava
- Wlogout
- Zathura
- Btop
- Neofetch
- Rustfmt
- Bash aliases and shell config
- User systemd services
- greetd config
- GRUB config
- package lists

## Goals

- keep the setup reproducible
- keep the repository clean and public
- avoid storing secrets or machine-specific private data
- make reinstalling Arch easier without over-automating everything
